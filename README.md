# 🌍 Wanderlog

A simple Trip Planner web application built using HTML, CSS and JavaScript.

## 🚀 Features
- Add new trips
- Edit existing trips
- Delete trips
- Save trips using LocalStorage
- Responsive design

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
- LocalStorage
